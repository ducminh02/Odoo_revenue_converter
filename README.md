# Odoo_revenue_converter
This is a customized add-on for Odoo Framework Customer Relationship Management (Ver 12). With this add-on the user can access the currencies faster
with a Menu instead of having to go through a long way of finding it.

This Exstension also enable the user to update any currencies (from EUR to currency) every hour/ day(or however the user wants) automatically using
a real time convertion API
